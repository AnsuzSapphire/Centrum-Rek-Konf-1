================

Repozytorium projektu Centrum Konferencyjno-Rekreacyjnego, PŁ 2013 sem. 5
================

Sugeruje dobrze zapoznać się z GitHubem, tutoriale na stronie są bardzo poręcznie i nie powinny sprawiać kłopotu ze zrozumieniem.
